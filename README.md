## 小紙條拼貼板（Python 版）

一個用 Python（Tkinter）寫的簡單桌面小工具，可以在大畫布上自由新增、拖曳、縮放便條紙，像在牆上拼貼便利貼一樣排版。

### 環境需求

- 已安裝 **Python 3.8+**
- Windows / macOS / Linux 皆可（系統要有 Tkinter，通常預設就有）

### 安裝與執行

1. 在此專案資料夾中打開終端機（或 PowerShell）  
2. 執行：

```bash
python main.py
```

看到視窗跳出來，就可以開始使用。

### 使用說明

- **新增便條**：按上方工具列的「新增便條」按鈕，或從「檔案」→「新增便條」
- **移動便條**：按住便條上方的標題列拖曳，就可以把便條拖到任何位置，自由拼接排列
- **修改內容**：直接在便條的文字區輸入或編輯文字
- **調整大小**：拖曳右下角的小方塊可以縮放便條尺寸
- **變更顏色**：點標題列右邊的「🎨」按鈕，可選擇便條背景顏色
- **刪除便條**：點標題列右邊的「✕」按鈕
- **儲存版面**：按「儲存版面」，會將所有便條的位置、大小、內容、顏色存成一個 `.json` 檔
- **載入版面**：按「載入版面」，選擇之前儲存的 `.json` 檔即可還原原本的拼貼排版
